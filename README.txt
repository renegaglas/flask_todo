this simple todo task manager is a Flask project.
it use an online monogo data base.

its main feature is that it is capble to regroupe memo into block (folder) for better managment.

you can creat a block or a memo.
for each block you have to define:
	- his name
for each memo you have to define:
	- his content
	- his importance (important, unimportant)
	- his parent block which you can select from every existing block

you can't update your block or memo but you can destroy them.
obviously if you destroy a block every memo link to this block will also be destroy.

this is everythings you can do with this todo app!
